# リッチ投稿ツール（個人用）

自分が管理者権限を持つLINEオープンチャット・グループ向けに、既存のFlex Message（リッチなカード型メッセージ）のJSONを貼り付けて、ボタンのリンク先URLなどを編集し、LIFF経由で自分のアカウントとして送信するための個人用ツールです。

**重要**: LIFFの`shareTargetPicker`（送信先を選ぶ方式）は**オープンチャットを送信先の選択肢に出せません**。オプチャに直接送るには`liff.sendMessages()`という別のAPIを使い、「そのオプチャの中に貼ったこのページのリンクを、オプチャの中からタップして開く」ことで、開いたそのトークに直接送信します。

## セットアップ（初回のみ）

1. [LINE Developers Console](https://developers.line.biz/console/) にログインし、プロバイダーを作成（または既存のものを使用）します。
2. チャネルを新規作成 → **LINEログイン** チャネルを選びます。
3. 作成したチャネルの「LIFF」タブから「追加」を押し、LIFFアプリを新規作成します。
   - **エンドポイントURL**: このツールをGitHub Pagesに公開した後のURL（例: `https://<username>.github.io/openchat-rich-tool/`）
   - **サイズ**: Full 推奨
   - **Scope**: `profile` と **`chat_message.write`** にチェック（`chat_message.write`が`liff.sendMessages()`に必要です。コンソール上に見当たらない/追加申請が必要な場合があります）
   - **Bot linkフィーチャー**: オフでOK
4. 作成後に表示される **LIFF ID**（`1234567890-AbCdEfGh` のような形式）をコピーします。
5. GitHub Pagesで公開したこのツールのページを開き、LIFF IDを入力して「保存して初期化」を押します。
6. 初回はLINEログインの許可画面が出るので許可します。

## 使い方（オプチャに直接送る場合）

1. このツールのURL（`https://<username>.github.io/openchat-rich-tool/`）を、送りたいオプチャの中に一度メッセージとして貼っておきます。
2. **そのオプチャの中から**、貼ったリンクをタップしてこのページを開きます（この開き方が重要。他の開き方だと直接送信できません）。
3. 送信したいFlex MessageのJSON（1件のオブジェクト、または複数まとめた配列）を「Flex MessageのJSON」欄に貼り付け、「読み込む」を押します。
4. 「ボタンのURLを編集」を開くと、JSON内の`action.type: "uri"`のボタンがラベル・URLごとに一覧表示されるので、リンク先を書き換えられます。
5. 「JSONを直接編集する」を開くと、変更後のJSON全体を直接編集・確認できます（編集後は「この内容を反映」を押すと再読み込みされます）。
6. プレビューで確認し、ボタンが「このトークに送信」になっていることを確認してから押します。そのオプチャに直接投稿されます。

上記の開き方をしなかった場合（LIFF ID再設定直後にそのまま送るなど）は、ボタンが「送信先を選んで送信」になります。この方式では**オプチャは選択肢に出ません**（1対1トークやグループのみ）。

## 注意

- LIFF IDはこのブラウザのlocalStorageにのみ保存されます。他の人には共有しないでください。
- このツールは自分専用です。他人に配布したり、権限のないトークへの投稿目的で使うことは意図していません。
